# 🤖 FS22 Discord Monitor Bot

Un bot Discord avanzato per monitorare server Farming Simulator 22 con notifiche in tempo reale, changelog delle mod e molto altro!

## ✨ Funzionalità

- 🔔 **Notifiche tempo reale** - Cambiamenti stato server e giocatori
- 📊 **Monitoraggio 24/7** - Controllo continuo del server
- 📦 **Changelog mod giornaliero** - Report automatico delle mod
- 👥 **Info giocatori** - Dettagli sui giocatori connessi
- 🎮 **Comandi user-friendly** - Interfaccia con emoji e embed
- ⚡ **Hosting gratuito** - Configurato per Replit + UptimeRobot

## 🚀 Installazione Rapida

1. **Clona su Replit**
   - Vai su [Replit](https://replit.com)
   - Clicca "Create Repl" → "Import from GitHub"
   - Incolla l'URL di questa repository

2. **Configura le variabili d'ambiente**
   - `DISCORD_TOKEN`: Il token del tuo bot Discord
   - Imposta i parametri in `config.json`

3. **Avvia il bot**
   - Clicca "Run"
   - Configura UptimeRobot per keep-alive

## 📋 Comandi Disponibili

- `/status` - Stato del server
- `/players` - Giocatori online
- `/mods` - Lista mod installate
- `/serverinfo` - Info dettagliate server
- `/help` - Guida comandi
